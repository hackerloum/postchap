# Blog Cover Image Prompts

Use these prompts to generate **1200×630 px** cover images for each article. Export as WebP or JPG and save under `public/blog/covers/[slug].webp`. ArtMaster brand: dark background (#0a0a0a / #111), accent yellow-green (#e8ff47), professional, minimal text (or none so OG looks clean).

**Style note for all:** Modern, flat or soft 3D, dark theme, lime/yellow-green accent, suitable for African small business and social media. No photorealistic faces. Prefer abstract shapes, devices, or simple scene composition.
